API Documentation

Create Applicant

Creates a new applicant with the provided data.

    
        POST /applicant
        Request Body:
        {
            "fullName": "John Doe"
        }
        Response:
        Status Code: 201 Created
        Body: "created applicant with id {id}"

Read Applicant

Reads an applicant by ID.

    
        GET /applicant/{id}
        Response:
        Status Code: 200 OK
        Body: {
            "id": 1,
            "fullName": "John Doe"
        }

Read Applicant List

Reads the list of all applicants.

    
        GET /applicant
        Response:
        Status Code: 200 OK
        Body: [
            {
                "id": 1,
                "fullName": "John Doe"
            },
            ...
        ]

Update Applicant

Updates an applicant by ID with the provided data.

    
        PUT /applicant/{id}
        Request Body:
        {
            "fullName": "Updated Name"
        }
        Response:
        Status Code: 200 OK
        Body: {
            "id": 1,
            "fullName": "Updated Name"
        }

Delete Applicant

Deletes an applicant by ID.

    
        DELETE /applicant/{id}
        Response:
        Status Code: 204 No Content
