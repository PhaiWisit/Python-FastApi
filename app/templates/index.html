API Documentation

Create Applicant

Creates a new applicant with the provided data.

    
        POST /applicant
        Request Body:
        {
            "fullName": "John Doe",
            "age": 25,
            "gender": "Male",
            "address": "123 Main St",
            "email": "john.doe@example.com",
            "phone": "123-456-7890",
            "expectedSalary": 50000
        }
        Response:
        Status Code: 201 Created
        Body: "created applicant with id {id}"

Read Applicant

Reads an applicant by ID.

    
        GET /applicant/{id}
        Response:
        Status Code: 200 OK
        Body: {
            "id": 1,
            "fullName": "John Doe",
            "age": 25,
            "gender": "Male",
            "address": "123 Main St",
            "email": "john.doe@example.com",
            "phone": "123-456-7890",
            "expectedSalary": 50000
        }

Read Applicant List

Update Applicant

Updates an applicant by ID with the provided data.

    
        PUT /applicant/{id}
        Request Body:
        {
            "fullName": "Updated Name",
            "age": 30,
            "gender": "Female",
            "address": "456 Oak St",
            "email": "updated.name@example.com",
            "phone": "987-654-3210",
            "expectedSalary": 60000
        }
        Response:
        Status Code: 200 OK
        Body: {
            "id": 1,
            "fullName": "Updated Name",
            "age": 30,
            "gender": "Female",
            "address": "456 Oak St",
            "email": "updated.name@example.com",
            "phone": "987-654-3210",
            "expectedSalary": 60000
        }

Delete Applicant

Deletes an applicant by ID.

    
        DELETE /applicant/{id}
        Response:
        Status Code: 204 No Content

Get CSV

Exports the list of applicants as a CSV file.

    
        GET /get_csv
        Response:
        Status Code: 200 OK
        Body: (CSV file download)

User Registration

Registers a new user with the provided username and password.

    
        POST /auth/register
        Request Body:
        {
            "username": "new_user",
            "password": "secure_password"
        }
        Response:
        Status Code: 200 OK
        Body: {
            "message": "User created successfully"
        }

User Login

Logs in a user and provides access and refresh tokens.

    
        POST /auth/login
        Request Body:
        {
            "username": "existing_user",
            "password": "existing_password"
        }
        Response:
        Status Code: 200 OK
        Body: {
            "access_token": "generated_access_token",
            "refresh_token": "generated_refresh_token"
        }

User Logout

Logs out a user by invalidating the provided access token.

    
        POST /auth/logout
        Authorization Header:
        Bearer {access_token}
        Response:
        Status Code: 200 OK
        Body: {
            "message": "Logout Successfully"
        }
